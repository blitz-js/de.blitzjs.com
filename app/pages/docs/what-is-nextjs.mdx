title: Was ist Next.js?
sidebar_label: Was ist Next.js?
---

Next.js ist in Blitz eingebettet und fast alle Funktionen von Next.js werden
unterstützt. Weitere Informationen zu den Unterschieden können hier gefunden werden:
[Warum Blitz anstelle von Next.js verwenden](https://blitzjs.com/docs/why-blitz)

[**Next.js**](https://nextjs.org/) ist ein bekanntes React Framework mit vielen
eingebauten Features wie beispielsweise:

- Pre-rendering, static generation \(SSG\) und server-rendering
  \(SSR\) werden auf einer per-page Basis unterstützt
- Automatisches Code-Splitting für schnellere Seitenladezeiten
- Clientseitiges Routing mit optimiertem Prefetching
- Entwicklungsumgebung, die Hot Module Replacement unterstützt
- Vollständig erweiterbar
